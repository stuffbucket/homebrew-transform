# homebrew-transform

GitHub Action to fix Homebrew formulas generated by GoReleaser.

## Problem

GoReleaser generates formulas with `def install` inside architecture blocks, which causes Homebrew lint errors:

```ruby
on_macos do
  on_arm do
    url "..."
    def install        # ❌ Lint error
      bin.install "foo"
    end
  end
end
```

## Solution

This action hoists method definitions to class level:

```ruby
on_macos do
  on_arm do
    url "..."
  end
end

def install            # ✅ At class level
  bin.install "foo"
end
```

## Usage

```yaml
name: Fix Formulas

on:
  push:
    paths:
      - 'Formula/*.rb'

permissions:
  contents: write

jobs:
  fix:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 2

      - name: Get changed files
        id: changed
        run: |
          files=$(git diff --name-only HEAD~1 HEAD -- 'Formula/*.rb' | tr '\n' ' ')
          echo "files=$files" >> $GITHUB_OUTPUT

      - uses: stuffbucket/homebrew-transform@v1
        if: steps.changed.outputs.files != ''
        with:
          files: ${{ steps.changed.outputs.files }}

      - name: Commit
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"
          git add -A
          git diff --staged --quiet || git commit -m "fix: transform formula" && git push
```

## Inputs

| Input | Description | Default |
|-------|-------------|---------|
| `files` | Space-separated formula files | `Formula/*.rb` |
| `config` | Path to custom transforms.yml | (built-in) |
| `ruby-version` | Ruby version | `3.3` |

## Outputs

| Output | Description |
|--------|-------------|
| `transformed` | List of files that were transformed |

## Configuration

The default transform hoists `install`, `test`, and `caveats` methods. To customize:

```yaml
# transforms.yml
transforms:
  - name: hoist_methods
    config:
      methods:
        - install
        - test
        - caveats
```

```yaml
- uses: stuffbucket/homebrew-transform@v1
  with:
    config: transforms.yml
```
